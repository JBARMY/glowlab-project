# GlowLab Beauty Group — Revenue Analysis Q1+Q2 2026

## Sobre o projeto
Este é um projeto fictício criado para fins de aprendizado em análise de dados com Python.
Os dados, empresa e personagens são completamente simulados.

## O que foi feito
- Geração de dataset simulado com 500 vendas
- Limpeza e validação dos dados
- Análise de KPIs financeiros
- Ranking de produtos por receita
- Análise de tendência mensal
- Dashboard executivo com 4 gráficos
- Relatório final em PDF

## Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- FPDF2
